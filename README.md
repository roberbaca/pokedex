## About The Project

Features:

- Show pokemon cards with basic info.
  
- Get more details selecting a specific pokemon and a pop-up window will show up.
  
- Pagination: the app shows only 10 pokemons per page.

- Search bar to search pokemons by name.

- Responsive design.
